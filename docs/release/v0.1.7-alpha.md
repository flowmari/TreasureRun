# TreasureRun v0.1.7-alpha — First-run setup guide refinement

This alpha release improves the first-time local playtest documentation by keeping the core setup route concise and moving optional checks to a separate reference page.

## Documentation

- Aligned the Architecture Guide headings with their technical-summary purpose.
- Added a linked six-step overview so the complete first-run route is visible before the detailed instructions.
- Focused the Player Quickstart on prerequisites, startup, joining, the first playtest, shutdown, and feedback.
- Moved screenshots and optional checks for heartbeat, the demo world, Treasure Shop, ranking/MySQL, and commands to a separate reference page.
- Clarified that the existing local startup command starts MySQL together with Spigot; no separate manual database path was added.
- Kept the core commands self-contained and copy-pasteable.

The setup-guide changes were tracked in #119 and #121 and implemented in #120 and #122.

## Scope

This release is limited to documentation structure and release metadata.

It does not change Java runtime behavior, gameplay logic, configuration behavior, i18n sources, language files, ResourcePack assets, database schemas, or ranking implementation.

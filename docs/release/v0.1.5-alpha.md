# TreasureRun v0.1.5-alpha — Heartbeat cue audible fix

This alpha follow-up release makes the proximity heartbeat cue easier to hear, test, and verify during a local playtest.

## Highlights

- Makes heartbeat cues easier to notice during playtests by using the `PLAYERS` sound category by default.
- Raises the default heartbeat volume for alpha testing.
- Adds `/heartbeatTest` as a quick operator-only sound check before running a full treasure hunt.
- Updates the Player Quickstart so testers know to check `Options > Music & Sounds > Players`.
- Keeps the heartbeat behavior configurable through the `heartbeat.*` settings in `config.yml`.

## Context

This release follows `v0.1.4-alpha`, which synchronized the Treasure Shop quickstart documentation.

The scope is intentionally narrow. This release does not introduce a new gameplay system; it records the already-merged heartbeat cue audible fix as a small, tester-facing alpha improvement.

## Boundaries

- Spigot 1.20.1 remains the verified target.
- This is still an early alpha release.

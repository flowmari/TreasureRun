# README Fresh Clone QuickStart Status Alignment

## Purpose

This documentation-only change aligns the README with the existing Fresh Clone QuickStart evidence.

The project already has documented Fresh Clone QuickStart PASS evidence, including one recorded local startup measurement. The README still contained older wording saying that a startup-time claim would only be added after measurement.

## Change

The README now points readers to:

`docs/external/FRESH_CLONE_QUICKSTART_EVIDENCE.md`

It states that one recorded local fresh-clone measurement completed startup successfully in 39s.

## Boundaries

This change does not:

- rewrite the alpha setup guide;
- change Java source code;
- change gameplay behavior;
- change DB migrations;
- restart Docker;
- mutate GitHub Releases;
- publish to SpigotMC or Paper;
- claim Paper compatibility;
- claim production/live-server readiness.

## Technical review value

This keeps the public README consistent with the repository evidence before asking an independent external tester to try the project.

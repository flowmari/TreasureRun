# i18n research logs archive

This directory preserves historical i18n review and batching notes.

These files are useful for traceability, but they are not part of the main first-time reader path. Current setup, gameplay, architecture, verification, and contribution guidance should remain easier to find from the main documentation path.

This archive does not define runtime behavior, language file content, ResourcePack assets, Fabric behavior, ranking API behavior, or Java implementation behavior.

# Security Policy

## Supported versions

TreasureRun is currently in early alpha development.

| Version | Supported |
|---|---|
| v0.1.0-alpha | Yes |
| earlier experimental commits | No |

## Reporting a vulnerability

Please do not publicly disclose security issues before they are reviewed.

If you find a vulnerability, please open a private report through GitHub Security Advisories if available, or contact the maintainer through the repository.

Please include:

- affected version or commit
- reproduction steps
- expected behavior
- actual behavior
- logs or screenshots if useful

## Scope

Security-relevant areas may include:

- server-side command handling
- packet handling
- ResourcePack delivery
- file generation
- database access
- unsafe assumptions around player-provided input
